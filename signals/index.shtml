JMRI: Victorian Railways Signaling Definition

This directory contains signaling definitions corresponding to common Victorian Railways double searchlight signalling practices. The reference used for these definitions is the Three Position signalling described on Mark Bau's website - http://www.victorianrailways.net/signaling/signalhome.html

See the aspect page.

Signal Mast definitions:

  • Searchlight - Two Head signal
  • Searchlight - Three Head signal
  • Searchlight - Single head Dwarf signal
